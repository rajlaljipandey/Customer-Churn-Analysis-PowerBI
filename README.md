# Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn patterns in a banking dataset using Power BI. The dashboard provides insights into customer attrition across different demographics, geographies, age groups, and product ownership.

## Tools Used

- Power BI
- DAX
- Power Query
- CSV Dataset

## Key Metrics

- Total Customers: 10,000
- Churned Customers: 2,037
- Retained Customers: 7,963
- Churn Rate: 20.37%

## Dashboard Features

- KPI Cards
  - Total Customers
  - Churned Customers
  - Retained Customers
  - Churn Rate

- Interactive Slicers
  - Gender
  - Geography
  - Age Group

- Visual Analysis
  - Churn by Gender
  - Churn by Geography
  - Churn by Age Group
  - Churn by Number of Products

## Key Insights

- Female customers showed higher churn compared to male customers.
- Germany had the highest customer churn among all regions.
- Customers aged 41–50 were more likely to churn.
- Customers with only one product showed the highest churn rate.
- Overall churn rate was 20.37%.

## Business Impact

The dashboard helps identify high-risk customer segments and supports customer retention strategies through data-driven decision making.
